About the project

It's functionalities include Login, Register, Create Recipe, Like and Dislike Recipe (for non-author of the recipe), Edit and Delete (for author of the recipe), see Profile Page of every user, Catalog with all recipes with filter options, Search by Recipe name or by author of recipe, Search for user by email.

It uses JWT for authentication, bcrypt to hash password on the back-end, it has validation for all forms on the front-end and on the back-end.
